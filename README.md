# Sampah-Dumai
Analisis progress timbulan dan pengurangan sampah Kota Dumai dalam Program Adipura KLH


# Analisis Sampah Kota Dumai

## Tujuan
Menganalisis kinerja pengelolaan sampah per kecamatan menggunakan indikator WLSRI.

## Data
- Sumber: Data 7 kecamatan Kota Dumai
- Format: Excel

## Metode
- WLSRI (Weighted Local Solid Waste Index)
- Analisis perbandingan antar kecamatan

## Tools
- Python (Google Colab)
- Visualisasi grafik

## Output
- Grafik perbandingan
- Ranking kecamatan
- Rekomendasi kebijakan

## Catatan
Repository ini digunakan sebagai pusat data dan analisis berkelanjutan.
